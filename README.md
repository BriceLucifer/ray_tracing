# Ray Tracing in Rust

1. 运行
```bash
cargo run > image.ppm
```
2. 编译
```bash
cargo build
```
3. file structure
```bash
    src
    ├── color.rs
    ├── hittable.rs
    ├── hittable_list.rs
    ├── lib.rs
    ├── main.rs
    ├── ppm.rs
    ├── ray.rs
    ├── rtweekend.rs
    ├── sphere.rs
    └── vec3.rs

1 directory, 10 files
```
